# Asset Library Workflow – Skill-Beschreibung

## Trigger

Verwende diesen Skill immer dann, wenn der User:

- Einen neuen Skill, ein Template oder eine Datei in die Asset Library eintragen möchte
- Formulierungen wie folgende verwendet:
  - „Leg das als Skill ab"
  - „Trag das in die Asset Library ein"
  - „Erstelle einen neuen Asset-Eintrag"
  - „Füge das zur Bibliothek hinzu"
  - „Push das auf GitHub"
  - „Speicher den Skill in Notion"

## Kompatibilität

Erfordert Notion MCP. GitHub MCP optional (sonst manueller Push via Terminal).

## Zugehörige Datei

SKILL.md enthält die vollständige Ausführungslogik inkl. Notion-IDs und GitHub-Struktur.
