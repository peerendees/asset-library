# Slack Daily Summary – Skill-Beschreibung

## Trigger

Verwende diesen Skill immer dann, wenn der User:

- Slack-Inhalte **zusammenfassen** möchte
- Inhalte **täglich protokollieren** oder in Notion übertragen möchte
- Formulierungen wie folgende verwendet:
  - „Was gab es heute in Slack?"
  - „Fasse #kanal zusammen"
  - „Tages-Update aus Slack"
  - „Slack-Digest"
  - „Slack nach Notion"
  - „Was wurde heute in Slack gepostet?"

## Kompatibilität

Erfordert **Slack MCP** + **Notion MCP**

## Zugehörige Datei

→ `SKILL.md` enthält die vollständige Ausführungslogik.
